String null 的串接

console output: nullworld
原因為String的串接是用下面的方式
test = String.valuOf(test)+ word
也就是當傳入的test為null時，會回傳”null” String回來

Vim Markdown Plugin

最近在安裝vim，發現網路上使用vbundle的方式安裝似乎沒有辦法成功，因此記錄一下最後怎麼安裝成功的

cd 至bundle clone vim-markdown，最後再把clone下來的ftplugin、ftdetect、syntax這三個資料夾複製到 ~/.vim/下。
